# Transfer-Learning
这是一个有关迁移学习的仓库，在这里可以看到迁移学习的各种用法。  
学习迁移学习可以帮助大家更好的理解什么是特征提取，什么是主干网络，将会对大家的神经网络学习产生很重要的影响。

# 仓库内容
该仓库会保存各种常用的特征提取网络，如VGG，mobilenet，mobilenetV2，resnet50，Xceptioin等。  
该仓库还会存储这些特征提取网络的应用，如猫狗分类、语义分割、目标检测等。
